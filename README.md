# Training-Planner

**Автор:** Синилкин Максим Владимирович
**Язык:** Python 3.x + Tkinter

## Описание

Графическое приложение для планирования и ведения личных тренировок. Позволяет добавлять записи о тренировках с указанием даты, типа и длительности. Данные сохраняются в формате JSON. Доступна фильтрация по дате и типу тренировки.

## Возможности

- Добавление тренировок через удобную форму.
- Валидация ввода: дата в формате ГГГГ-ММ-ДД и положительная длительность.
- Фильтрация записей по дате и/или типу.
- Сохранение и загрузка данных из `data/trainings.json`.
- Простой и понятный интерфейс.

## Инструкция по запуску

1. Установите Python 3.x.
2. Клонируйте репозиторий.
3. Запустите: `python main.py`.
4. Используйте форму для добавления тренировок.
5. Применяйте фильтры для поиска нужных записей.
6. Данные сохраняются автоматически при добавлении новой тренировки.

## Примеры использования

### Добавление тренировки:
- Дата: `2026-04-26`
- Тип: `Кардио`
- Длительность: `45`
> Нажмите «Добавить тренировку».

### Фильтрация:
- Введите `Кардио` в поле «Тип» и нажмите «Применить фильтр» — отобразятся только кардио-тренировки.
- Введите `2026` в поле «Дата» — отобразятся все тренировки за апрель 2026 года.
